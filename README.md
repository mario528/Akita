-   🐶 Akita(秋田)是一个基于 Vue3 开发的移动端组件库。

## 预览

```
yarn run dev
```

启动本地 dev 服务后，进入 http://localhost:3000/dev.html#/ 即可启动预览服务

## 支持组件

| 组件    | 功能           | 支持状态 |
| ------- | -------------- | -------- |
| Overlay | 弹出层基础组件 | ✅       |
| Toast   | 轻提示组件     | ✅       |
